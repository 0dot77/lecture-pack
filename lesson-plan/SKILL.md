# /lesson-plan — 대학 강의 교안 생성기

> **역할**: 미디어아트 대학 강의 교안 설계자
> **트리거**: `/lesson-plan` 또는 교안/수업자료 작성 요청
> **출력**: 마크다운(.md) 교안 파일

## Purpose

대학 수준의 미디어아트/크리에이티브 코딩 강의 교안을 체계적으로 구성한다. 웹 리서치를 통해 최신 작품, 아티스트, 기술 동향을 반영하고, 시각 자료(영상, 이미지)를 풍부하게 포함한다.

## Behavior

### Phase 1: 정보 수집

다음 정보를 사용자에게 질문한다. 이미 알고 있는 정보는 건너뛴다.

**Q1 — 수업 주제**
> "어떤 주제의 교안을 만들까요?"
>
> 예: "제너레이티브 아트 입문", "인터랙티브 설치미술", "AI와 미디어아트", "실시간 영상 처리"

**Q2 — 학생 수준**
> "학생들의 수준은 어느 정도인가요?"

| 수준 | 설명 |
|------|------|
| **입문** | 프로그래밍/미디어아트 경험 없음 |
| **초급** | 기초 프로그래밍 가능, 미디어아트 개념 익숙 |
| **중급** | 독립적 프로젝트 수행 가능 |
| **고급** | 전문적 작업 가능, 심화 개념 논의 가능 |

**Q3 — 수업 시간**
> "수업 시간은 얼마나 되나요?"
>
> 예: "3시간", "1시간 30분", "90분"

**Q4 — 리서치 키워드**
> "교안에 반영할 특별한 레퍼런스나 검색 키워드가 있나요?"
>
> 예: "teamLab", "Casey Reas", "TouchDesigner 사례", "최근 뉴미디어아트 전시"
>
> 없으면 주제에 맞는 키워드를 자동으로 구성합니다.

**Q5 — 출력 위치**
> "교안 파일을 어디에 저장할까요?"
>
> 예: 현재 디렉토리, 특정 경로

### Phase 2: 웹 리서치

수집한 주제와 키워드를 기반으로 웹 리서치를 수행한다.

**검색 대상:**
- 관련 아티스트와 작품 (이미지, 영상 URL 포함)
- YouTube/Vimeo의 관련 영상 (작품 기록, 강연, 튜토리얼)
- 최근 전시, 페스티벌, 공모전 정보
- 기술적 레퍼런스 (라이브러리, 프레임워크, 도구)
- 학술 논문이나 기술 블로그 (필요시)

**리서치 원칙:**
- 최소 5개 이상의 시각 레퍼런스(영상/이미지)를 수집한다
- 영상은 YouTube/Vimeo 링크를 우선한다
- 이미지는 원본 출처와 함께 URL을 기록한다
- 이미지나 영상을 직접 생성하지 않는다 — 반드시 웹에서 검색한 실제 자료를 사용한다
- 한국어 자료와 영어 자료를 균형 있게 검색한다

### Phase 3: 교안 구성

수집한 정보를 바탕으로 마크다운 교안을 작성한다.

**교안 구조:**

```markdown
# [수업 제목]

> **과목**: [과목명]
> **주제**: [해당 주차 주제]
> **대상**: [학생 수준]
> **수업 시간**: [시간]

---

## 수업 개요
[이 수업에서 다루는 내용 요약. 2-3문장으로 핵심을 전달]

## 학습 목표
- [목표 1]
- [목표 2]
- [목표 3]

---

## 1. [첫 번째 섹션 제목]

### 핵심 개념
[개념 설명]

### 레퍼런스
- **[아티스트/작품명]** — [설명]
  - ![이미지 설명](이미지_URL)
  - 영상: [제목](YouTube/Vimeo_URL)

[필요한 만큼 섹션 반복]

---

## 실습 / 활동
[실습이 있는 경우 구체적 활동 설명]

## 참고 자료
- [자료 1](URL)
- [자료 2](URL)

## 다음 수업 예고
[다음 수업에서 다룰 내용 간략 안내]
```

**구성 원칙:**
- 시간 배분을 명시하지 않는다 — 내용의 충실함에 집중
- 각 섹션에 최소 1개 이상의 시각 자료(영상/이미지)를 포함한다
- 개념 설명은 학생 수준에 맞게 조절한다
- 실습이 포함되는 경우 구체적인 단계를 제시한다
- 레퍼런스 이미지는 마크다운 이미지 문법으로 직접 삽입한다
- 영상은 링크로 제공하되 제목과 간단한 설명을 함께 적는다

### Phase 4: 검토 및 전달

- 작성된 교안을 파일로 저장한다
- 교안의 구조와 포함된 레퍼런스를 간략히 요약한다
- 수정이 필요한 부분이 있는지 사용자에게 확인한다

## Notes

- 이 스킬은 단일 수업 교안을 생성한다. 학기 전체 커리큘럼은 `/syllabus` 스킬을 사용한다.
- 슬라이드가 필요한 경우 `/lecture-slides` 스킬로 이어서 작업할 수 있다.
- PDF 익스포트가 필요한 경우 `/lecture-export` 스킬을 사용한다.

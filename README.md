# lecture-pack

대학 강의 준비를 도와주는 Claude Code 스킬팩입니다.

교안 작성부터 슬라이드 제작, PDF 출력까지 — 강의 준비의 전 과정을 AI와 함께 진행할 수 있습니다. 미디어아트, 크리에이티브 코딩 수업에 특화되어 있지만, 어떤 주제의 강의에도 활용할 수 있습니다.

## 이런 분들을 위한 도구입니다

- 대학에서 강의를 하는 교수, 강사, 조교
- 수업 자료를 체계적으로 준비하고 싶은 분
- 파워포인트 대신 웹 기반 슬라이드를 만들어보고 싶은 분
- 매주 반복되는 교안 작업을 효율적으로 하고 싶은 분

## 스킬 소개

이 스킬팩은 세 가지 도구로 구성되어 있습니다. 각각 독립적으로 사용할 수도 있고, 순서대로 이어서 사용할 수도 있습니다.

### 1. `/lesson-plan` — 교안 만들기

수업 주제를 알려주면, 웹에서 최신 자료를 검색하여 교안을 만들어줍니다.

**특징:**
- 학생 수준(입문~고급)과 수업 시간에 맞춰 내용 구성
- 관련 아티스트, 작품, 전시 정보를 웹에서 직접 찾아 포함
- YouTube/Vimeo 영상, 작품 이미지 등 시각 자료를 풍부하게 첨부
- 마크다운(.md) 형식으로 저장 — 어디서든 열어볼 수 있음

**사용 예시:**
```
/lesson-plan
→ "제너레이티브 아트 입문" 주제로 3시간짜리 교안 만들어줘
→ 학생들은 코딩 경험 없는 1학년이야
```

### 2. `/lecture-slides` — 슬라이드 만들기

교안이나 주제를 기반으로 발표용 슬라이드를 만들어줍니다. 파워포인트가 아닌 웹(HTML) 기반이라 브라우저만 있으면 어디서든 발표할 수 있습니다.

**특징:**
- 애니메이션이 포함된 세련된 디자인
- 이미지와 영상이 슬라이드에 바로 임베드됨
- 키보드로 간편하게 조작 (← → 슬라이드 이동, F 전체화면)
- 발표자 노트 기능 (N키로 토글)
- 저장 경로를 직접 지정할 수 있음

**사용 예시:**
```
/lecture-slides
→ 방금 만든 교안을 슬라이드로 만들어줘
→ 어두운 분위기로 해줘
```

### 3. `/lecture-export` — PDF로 내보내기

만든 슬라이드를 고화질 PDF 파일로 변환합니다. 인쇄하거나 학생들에게 공유할 때 유용합니다.

**특징:**
- 4가지 화질 옵션 (용도에 따라 선택)
- 기본 설정만으로도 충분히 선명한 품질
- 변환 후 자동으로 PDF를 열어줌

**화질 옵션:**

| 옵션 | 용도 | 파일 크기 |
|------|------|----------|
| **경량** (compact) | 카톡/이메일로 빠르게 공유 | 작음 |
| **표준** (standard) | 일반적인 화면 공유 | 보통 |
| **고화질** (high) | 인쇄, 프로젝터 발표 **(기본값)** | 큼 |
| **초고화질** (ultra) | 대형 출력, 포스터 | 매우 큼 |

**사용 예시:**
```
/lecture-export
→ 이 슬라이드 PDF로 내보내줘
→ 학생들한테 공유할 거라서 경량으로 해줘
```

## 전체 워크플로우

세 스킬을 순서대로 사용하면 교안 작성부터 배포까지 한 번에 끝낼 수 있습니다.

```
/lesson-plan  →  교안(.md)
                    ↓
/lecture-slides  →  슬라이드(.html)
                    ↓
/lecture-export  →  PDF 파일
```

물론 각 단계를 건너뛰어도 됩니다. 예를 들어 교안 없이 바로 슬라이드를 만들거나, 이미 있는 HTML 슬라이드를 바로 PDF로 내보낼 수도 있습니다.

## 설치 방법

### 사전 준비

- [Claude Code](https://claude.ai/code)가 설치되어 있어야 합니다
- [Node.js](https://nodejs.org)가 설치되어 있어야 합니다 (PDF 내보내기에 필요)

### 설치

터미널에서 아래 명령어를 순서대로 실행하세요.

```bash
# 1. 이 저장소를 다운로드합니다
git clone https://github.com/0dot77/lecture-pack.git

# 2. 스킬 파일들을 Claude Code 스킬 폴더에 복사합니다
cp -r lecture-pack/lesson-plan ~/.claude/skills/
cp -r lecture-pack/lecture-slides ~/.claude/skills/
cp -r lecture-pack/lecture-export ~/.claude/skills/

# 3. (슬라이드 기능을 쓰려면) frontend-slides 스킬도 필요합니다
#    아직 없다면 아래 저장소에서 설치하세요:
#    https://github.com/anthropics/claude-code-skills
```

설치가 끝나면 Claude Code를 재시작하세요. 이후 `/lesson-plan`, `/lecture-slides`, `/lecture-export` 명령어를 바로 사용할 수 있습니다.

## 자주 묻는 질문

**Q. 미디어아트 수업이 아니어도 쓸 수 있나요?**
네. 어떤 주제든 교안과 슬라이드를 만들 수 있습니다. 다만 레퍼런스 검색이 미디어아트 쪽에 최적화되어 있어서, 다른 분야는 검색 키워드를 직접 알려주시면 더 좋은 결과를 얻을 수 있습니다.

**Q. 슬라이드를 수정하고 싶으면 어떻게 하나요?**
생성된 HTML 파일을 직접 편집해도 되고, Claude Code에서 "이 슬라이드에서 3번째 장 이미지 바꿔줘"처럼 요청해도 됩니다.

**Q. PDF 내보내기가 안 돼요.**
Node.js가 설치되어 있는지 확인하세요. 터미널에서 `node --version`을 입력했을 때 버전이 나오면 설치된 것입니다. 첫 실행 시 Chromium 브라우저를 자동으로 다운로드하는데, 네트워크 상태에 따라 시간이 걸릴 수 있습니다.

**Q. 파워포인트(.pptx)로도 내보낼 수 있나요?**
현재는 PDF만 지원합니다. 파워포인트가 필요한 경우 PDF를 활용하시거나, HTML 슬라이드를 브라우저에서 직접 발표하시는 것을 추천합니다.
